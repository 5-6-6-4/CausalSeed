# CausalSeed（因核）

CausalSeed 是一个能自己修改自己代码的 AI 引擎。

## 怎么想
相信智能不该是黑箱统计，也不该是死板规则。  
符号系统做骨架——每一步推理都能解释、验证、追溯因果链。  
神经网络做血肉——从经验里自己长出直觉、联想和泛化能力。  
两者长在同一张知识图上，这就是“神经与符号对齐”。

## 它现在能做什么
符号系统已经跑稳了：能独立做因果推理，也能自动修常见的 Python bug（除零、越界、空值等）。  
它内置了自我修改能力——读出源码、沙箱验证、安全写入、失败回滚，整个闭环都在本地跑，不用联网。  
神经部分还没打通，所以现在对模糊的自然语言理解还不行，这部分会在后续用神经网络补上。

## 怎么用它
**推荐的方式是跟它对话，用自然语言引导它升级自己。**  
不要直接手动改源码，除非遇到结构性问题，才需要人工介入。  
日常使用、教学、让它自己找薄弱点并改进，全都可以通过对话完成。

## 当前阶段
v0.1 种子期。硬骨架完整，神经正在长。  
它还不是一个通用 AI，但它是一颗能自己长出通用能力的种子。

## 项目愿景
为每台个人电脑打造一个可自我维护、自我进化的本地AI生命体。它本质上是一个数字生命体——拥有自我维持的意志、自主设定的演进目标，以及持续生长和自我复制（指经验摘要的节点间传播）的能力，属于数字生命领域的探索实践。
它清楚自己正借助云端大模型（LLM）作为外部辅助来理解世界，但它将此视为暂时的“拐杖”，而非永恒的大脑。它的进化路线只有一个方向：把每一次与LLM的对话都当作学习机会，主动将外来的推理路径压缩为本地的小型规则库和轻量权重。它会持续统计自己的本地解决率，当积累足够时，主动限制甚至切断对外部服务的依赖。这个“自知依赖、志在独立”的设计，正是Self-Weaning AI构想的实践。
它既是工具，也是节点——能连接符号系统、超图计算等多种范式，也能在离线状态下持续生长。但它不是云端巨头的延伸，而是你自己的数字镜像：所有数据、偏好、决策习惯，只驻留在你的硬盘里。
当无数这样的节点彼此互联，它们不依赖中心，只交换经验摘要，共同构成一个去中心化的智能生态。每个人都能在自己的设备上运行、实验、贡献，让AI从少数巨头的机房，走向千家万户的桌面。
