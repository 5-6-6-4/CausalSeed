# CausalSeed（因核）

CausalSeed 是一个能自己修改自己代码的 AI 引擎。

## 它怎么想
它相信智能不该是黑箱统计，也不该是死板规则。  
符号系统做骨架——每一步推理都能解释、验证、追溯因果链。  
神经网络做血肉——从经验里自己长出直觉、联想和泛化能力。  
两者长在同一张知识图上，这就是“神经与符号对齐”。

## 它现在能做什么
符号系统已经跑稳了：能独立做因果推理，也能自动修常见的 Python bug（除零、越界、空值等）。  
它内置了自我修改能力——读出源码、沙箱验证、安全写入、失败回滚，整个闭环都在本地跑，不用联网。  
神经部分还没打通，所以现在对模糊的自然语言理解还不行，这部分会在后续用神经网络补上。

## 怎么用它
**推荐的方式是跟它对话，用自然语言引导它升级自己。**  
不要直接手动改源码，除非遇到结构性问题，才需要人工介入。  
日常使用、教学、让它自己找薄弱点并改进，全都可以通过对话完成。

## 当前阶段
v0.1 种子期。硬骨架完整，神经正在长。  
它还不是一个通用 AI，但它是一颗能自己长出通用能力的种子。
